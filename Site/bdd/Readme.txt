 Instructions pour créer la base de données :
    - Connetectez vous à votre base de données à l'aide de votre SGBD de préférence
    - Utilisez la commande 'SOURCE <chemin vers fichier sql>' pour charger le fichier projet.sql
    - Mettez à jour vos identifiants dans le fichier 'param.inc.php'
    
Vous pouvez à présent utiliser le site.
